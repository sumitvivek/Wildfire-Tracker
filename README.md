# Wildfire Tracker

> This is a React app to track wildfires and plot them on a Google map.

## Usage

```
npm install
npm start

# Build
npm run build
or
yarn build
```

## API Keys

You will need to add your Google Maps API key in the Map.js component and register for a NASA API key at https://api.nasa.gov.
